# IUT_Arena

Version de la borne IUT de Calais qui contient le jeu IUT Arena créé dans le cadre du cours de Maintenance Applicative (BUT 3).  

## Auteurs
Manon Chagot et Quentin Douilly  
Etudiants en BUT3 INFO apprentis à l'IUT de Calais


## Description du jeu
Il s'agit d'un jeu de combat en solo ou à deux joueurs.  
Le but et de **pousser** ses adversaires en dehors de l'arène grâce à une **charge**,
et d'éviter de tomber avec la capacité d'**esquive**.  
  
En mode **solo** vous affronterez plusieurs vagues d'ennemis de plus en plus difficiles.  
En **duo** vous vous affronterez durant trois manches faire tomber l'adversaire vous donne un point.  

Que le meilleur gagne !

## Code source & lancement
Le code source de ce projet se trouve dans `/projet/IUT_Arena`.
Pour lancer la borne (sous Linux) :  
 - placez vous dans le dossier `borne_arcade`
 - exécutez le script bash avec la commande `./compilation.sh`
 - exécutez la commande `java Main`

*Pour plus d'information sur le lancement de la borne, se référer au fichier `borne_arcade/README.md`.*
